Session
Header

A logo must be a hosted URL to travel in a shared link. The channel name and connection dot always sit on the right.

Surfaces

Guests land on the first enabled surface; use the arrows to reorder. The tab bar appears automatically when more than one is enabled.

Pads
Label (shown)Key

Each pad sends one key the host maps to a sample (a single character, e.g. q). The label is just what the guest sees.

Keys (piano)

Note numbers, 0–127 (60 = middle C / the host’s root pitch).

Seq

A client-side sequencer that fires these keys at the tempo. A whole number is one pitched note (60 64 67); a single character is a pad key.

Send hints

Flags sent with every press, asking the host to randomize samples or hold pitched notes.

Colors

Covers the whole controller, header included: title/channel use Text, the status text uses Muted, and the header rule uses Borders. Colors travel fine in a shared link. Off by default (uses the built-in dark theme).

Advanced · Backend

Client Firebase keys aren’t secrets — they only identify which backend to reach. Prefilled from the host when you open this from the app. Leave as-is for the default backend.
