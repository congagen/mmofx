
# MMOSFX



#### Testing:


* HTTPS (MAC):
```sh
brew install nss
npm i -g --only=prod https-localhost
serve .
-> https://localhost
```

* HTTPS (LINUX):
```sh
sudo apt install libnss3-tools
sudo yum install nss-tools
sudo pacman -S nss
-> https://localhost
```
